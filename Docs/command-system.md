# Command System

## Core Commands

### /teamup
Find teammates.

### /create-project
Create private workspace.

### /summarize-paper
Generate AI summary.

### /research-ideas
Suggest project ideas.

### /daily-log
Submit daily progress.

### /leaderboard
Show rankings.

### /set-standup
Configure standup schedule.

## Cooldowns
Commands use cooldowns to prevent spam.
